# 500 AI Companies - Exploratory Data Analysis (EDA) 🤖📊

Hey there! Welcome to my project. I took a dataset of 500 top AI companies and did a clean, straightforward Exploratory Data Analysis (EDA) to find out where the AI industry is heading. 

## 🔗 Live Interactive Notebook
You can check out my full code, data cleaning steps, and interactive charts right here on Kaggle:
👉 **[Kaggle Notebook Link](https://www.kaggle.com/code/mujtabashokat/500-ai-companies-analyses/notebook)**

## 📌 What's Inside?
I wanted to keep things simple and focus on finding quick, valuable insights. Here is what I explored:
* **Geography:** Which countries and cities are the biggest AI hubs?
* **Money Matters:** How much funding are these startups raising and what are their valuations?
* **Top Sectors:** Which industries (like Healthcare, Finance, or SaaS) are using AI the most?

## 🛠️ Tools Used
Just the classic Python stack for data analysis:
* **Pandas & NumPy** for cleaning the data.
* **Matplotlib & Seaborn** for clean, static plots.
* **Plotly** for interactive visualizations.
